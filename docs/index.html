What is Huft?

Huft is a translation layer for .onion domains. It turns complicated, unreadable addresses like http://expyuzz4wqqyqhjn.onion into a human-readable list of words like web+huft://page-shower-samples-higher-someone-hire-bound, and vice versa.

How do I install Huft?

You can simply click here. The browser will display a confirmation dialog; click OK, and you're done.

How do I use Huft?

Once it's installed, you can simply click on web+huft:// links and they will work automatically. Try it here: it should redirect you to the homepage of the Onion project.
